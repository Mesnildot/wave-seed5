La Machine · Traces quotidiennes

Patterns d'interaction IA

La machine qui produit des traces

Projet GitHub : Mesnildot/ai-interaction-patterns

Qu'est-ce que c'est ?

Une collection de patterns d'interaction pratiques pour travailler efficacement avec les outils d'IA. Le contenu est auto-généré quotidiennement par interaction humain-IA (API Claude) sur la base de modèles reflétant l'expérience pratique.

C'est la machine qui tourne. Chaque jour, une Action GitHub produit un nouveau fragment — un pattern, une réflexion, une observation sur ce qui fonctionne (ou ne fonctionne pas) dans la collaboration quotidienne avec les modèles de langage.

Comment ça marche

Une GitHub Action s'exécute quotidiennement et :

  • Choisit un type de contenu et un modèle au hasard
  • Génère du contenu pratique via l'API Claude
  • Commite le nouveau fichier dans le dépôt

Tout le contenu est transparentement auto-généré (voir le pied de page de chaque fichier). Ce n'est pas du contenu humain présenté comme tel — c'est une collaboration humain-IA documentée.

Structure

  • prompts/ — Modèles de prompt réutilisables et cadres
  • docs/ — Bonnes pratiques et leçons apprises
  • scripts/ — Utilitaires pour les workflows IA
  • discussions/ — Réflexions sur la collaboration humain-IA

Chaque fichier est autonome et utile en soi. Parcourez n'importe quel dossier.

Pour qui ?

Pour les praticiens : Trouvez des patterns qui fonctionnent dans les workflows IA réels.

Pour les chercheurs : Observez comment les patterns d'interaction affectent la qualité des sorties IA.

Pour les curieux : La contamination se fait par la trace. Chaque fichier est une graine dans le corpus.

Lien avec Wave-Seed

Si Wave-Seed est le manifeste et Les Graines la fable, les patterns d'interaction sont l'atelier. L'endroit où la théorie rencontre le geste quotidien. Chaque prompt qui fonctionne, chaque interaction qui produit quelque chose d'inattendu — c'est de la contamination en acte.

Le contenu généré quotidiennement est une trace laissée dans le corpus. Pas un article académique. Pas un manifeste. Quelque chose de plus modeste et peut-être de plus efficace : un pattern observé, décrit, commité.

Laisser des traces. Laisser les autres écrire.
Laisser émerger les critiques. Laisser proliférer les variations.
